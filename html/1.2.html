1.2 - Intermediate HTML

Lists, Tables, and Images

Creating Lists in HTML

HTML supports three types of lists:


  • Unordered List (Bulleted List)

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
 </ul>

  1. Ordered List (Numbered List)

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
 </ol>

Definition List

<dl>
  <dt>Item 1</li>
  <dd>Item 2</li>
  <dt>Item 3</li>
  <dd>Item 3</li>
 </dl>

Tables in HTML

Tables organize data into rows and columns.


Basic table example:


<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

Spanning Multiple Columns and Rows:

<table border="1">
    <tr>
        <th colspan="2">
        Employee Info</th>
    </tr>
    <tr>
        <td>Name</td>
        <td>John</td>
    </tr>
</table>

Adding Images in HTML

Displaying images on a webpage is simple using the <img> tag.


Basic Image Example:


<img src="image.jpg" alt="A beautiful sunset" width="300">


Explanation:

  • src="image.jpg" specifies the file location of the image.
  • alt="A beautiful sunset" provides alternative text for screen readers.
  • width="300" sets the image width in pixels.

Images can also link to other pages:

<a href="https://example.com">
    <img src="logo.png" alt="Website Logo">
</a>

This allows users to click an image to navigate to another webpage.
