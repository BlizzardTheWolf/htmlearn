1.1 - Understanding HTML

The Basics & Easy Guide

What is HTML?

HTML, which stands for HyperText Markup Language, is the standard language used to create web pages. It provides the basic structure of a website by using a system of elements and tags to define content such as text, images, videos, and links.

Unlike programming languages, HTML does not perform logic or calculations; instead, it structures information so that browsers like Google Chrome, Firefox, and Safari can interpret and display it properly.

Why is HTML important?

HTML is fundamental to web development because it:

  • Provides the foundation for all web pages.
  • Allows content to be structured into readable and accessible formats.
  • Supports multimedia elements like images, videos, and audio.
  • Enables navigation through links and interactive forms.

When building a website, HTML works alongside CSS and JavaScript:

  • HTML defines the structure and content.
  • CSS applies styles such as colors, fonts, and layouts.
  • JavaScript adds interactivity, such as animations and dynamic elements.

Writing your first webpage

An HTML document consists of a hierarchy of elements enclosed in tags.

  • <!DOCTYPE html> declares the document as HTML5.

  • <html> is the root element containing all HTML code.

  • <head> stores metadata like the title and character encoding.

  • <title> defines the page title displayed in the browser tab.

  • <body> holds all visible content, including text, images, and links.

  • <h#> represents a heading.

  • <p> creates paragraphs.

  • <a> is a hyperlink, allowing navigation between pages. Use “href” within the tag to define the link.

Code:

<!DOCTYPE html>
 <html>
   <head>
     <title>My First Webpage</title>
   </head>
   <body>
     <h1>Welcome to My Website!</h1>
     <p>This is a paragraph of text.</p>
     <a href="https://www.example.com">Click here to visit another site</a>
   </body>
 </html>

Result:

To see your own result, follow these steps:


  • Open a Text Editor – Use Notepad, VSCode, or any HTML editor.

  • Write Your HTML Code – Save the file with a .html extension.

  • Open in a Browser – Double-click the file to view your webpage.
