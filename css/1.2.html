1.2 - Intermediate CSS

Selectors, Box Model, Flexboxes

Introduction to Intermediate CSS:

Intermediate CSS concepts provide developers with the tools to craft well-structured, flexible, and visually engaging web pages. At this stage, developers delve deeper into the power of CSS by mastering advanced selectors, which allow for precise targeting of HTML elements, enhancing both efficiency and maintainability. The box model becomes an essential foundation, helping developers understand how margins, borders, padding, and content interact to create visually consistent designs. Additionally, flexbox emerges as a powerful layout system, enabling developers to efficiently align, distribute, and position elements within containers, making responsive design more intuitive and effective. Mastering these concepts paves the way for creating dynamic, polished, and professional-looking websites.

Common CSS Selectors:

  • Type Selector: Styles all instances of an element. (p, h1, h2, h3, button)
  • Class Selector (.): Applies styles to multiple elements with the same class. (class="")
  • ID Selector (#) Targets a single, unique element. (id="")
  • Group Selector (,): Used to divide elements to give them all the same style.
  • Universal Selector (*): Applies styles globally.

Understanding the CSS Box Model:

div {
  width: 250px;
  padding: 15px;
  border: 2px dashed gray;
  margin: 20px;
  box-sizing: border-box;
}


Explanation:

  • width: Specifies the width of the element.
  • padding: Space between content and the border.
  • border: Outlines the padding and content.
  • margin: Space outside the border, separating the element from others.
  • box-sizing: Specifies what it included in the elementâ€™s total width and height.

Flexbox for Responsive Layouts:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 15px;
  flex-wrap: wrap;
}

.flex-item {
  padding: 10px;
  background-color: lightblue;
  flex: 1;
  text-align: center;
}


Explanation:

  • display: Specifies how everything is displayed within that container (flex, block, grid...)
  • justify-content: Aligns items horizontally.
  • align-items: Aligns items vertically.
  • text-align: Aligns element to the top, bottom, left, middle or right.
  • gap: Spaces between items in a flexbox.
  • flex: Controls item growth and shrinkage.
  • flex-wrap: Manages item wrapping for responsiveness.
