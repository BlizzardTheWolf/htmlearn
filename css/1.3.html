1.3 - Advanced CSS

Grid, Animations, Transitions, Variables

Animations and Transitions:

@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

.button {
  transition: background-color 0.5s, transform 0.3s;
}

.button:hover {
  animation: bounce 1s;
  transform: scale(1.1);
}

Explanation:

  • @keyframes: Defines the animation sequence.
  • transition: Smoothens property changes.
  • hover: Styles elements on mouse hover.
  • transform: Scales or moves elements.

Responsive Design with Media Queries:

@media (max-width: 768px) {
  body {
    font-size: 16px;
  }
  .grid-container {
    grid-template-columns: 1fr;
  }
  .button {
    padding: 12px;
  }
}

Explanation:

  • @media: Applies styles based on screen size.
  • grid-template-columns: Adjusts layout for small screens.
  • font-size: Enhances readability.
  • padding: Optimizes touch-friendly targets.

CSS Variables for Consistent Styling:

:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size-base: 16px;
}

body {
  color: var(--primary-color);
  font-size: var(--font-size-base);
}

Explanation:

  • :root: Declares global CSS variables.
  • var(): Applies variables for consistent styling.
  • Scalability: Variables simplify style management across large projects.
