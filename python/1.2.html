1.1 - Intermediate Python

Data Structures, Functions and more

Introduction to Intermediate Python

Intermediate Python concepts provide developers with the skills to write more efficient, scalable, and maintainable code. At this level, programmers delve into essential topics such as data structures, functions, and object-oriented programming (OOP), each of which plays a crucial role in building robust applications. Understanding data structures—such as lists, tuples, sets, and dictionaries—enables developers to store and manipulate data effectively, optimizing performance for various tasks. Functions help organize code into reusable blocks, improving readability and reducing redundancy, while concepts like recursion and higher-order functions add greater flexibility to programming logic. Object-oriented programming introduces principles such as classes, inheritance, polymorphism, and encapsulation, allowing developers to structure their code in a modular and reusable way. Mastering these intermediate Python concepts empowers developers to write cleaner, more efficient, and well-structured programs, paving the way for tackling more advanced topics and real-world software development challenges.

Python Data Structures:

  • Lists: Ordered, mutable collections.
  • Tuples: Ordered, immutable collections.
  • Sets: Unordered collections of unique elements.
  • Dictionaries: Key-value pairs for efficient lookups.
  • Deque (from collections): A double-ended queue for efficient appends/pops.

my_list = [1, 2, 3]
my_tuple = (1, 2, 3)
my_set = {1, 2, 3}
my_dict = {"key": "value"}

Functions in Python:

Object-Oriented Programming (OOP):

class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

  def greet(self):
    return: f"Hi, I'm {self.name} and I'm {self.age} years old."


person = Person("Bob", 25)
print(person.greet())

  • class: Defines a class.
  • init(): Initializes instance variables.
  • self: Refers to the instance of the class.
  • Inheritance: Allows classes to inherit attributes and methods from other classes.
